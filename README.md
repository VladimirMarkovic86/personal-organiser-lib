# Personal organiser library

Personal organiser library is clojure project that implements mechanism which calculates daily meal sizes for particular organism.

### Installing

You can use this project as dependency in clojure projects by listing it in project.clj

```
[org.clojars.vladimirmarkovic86/personal-organiser-lib "0.1.0"]
```
